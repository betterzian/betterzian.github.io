## **Projects**


### **1、Terminal cloud service business-feature-aware scheduling technology**
Huawei Technologies Co., Ltd. and Southeast University
#### Project Overview
Many enterprises will provide end users with services in the form of containers for longterm deployment in the cloud service provider, these services used resources have a tidal effect, in the resource valley, server resource utilization is not high, cloud service providers will be short-term services such as machine learning and other tasks into these servers to improve resource utilization.
#### Main responsibilities
1, Clearly decouple the problem and split it into multiple sub-problems. 2, A large number of reading papers, and combined with the needs, the establishment of a scheduler framework.3, The use of the LSTM model and transformer model for resource prediction.4, The proposed heuristic method and deep reinforcement learning algorithm for container scheduling, compared to the traditional algorithm improve nearly 10% of resource utilization.

### **3、Kubernetes scheduling algorithm optimization**
China Aerospace Science and Technology Co. 8511 Research Institute and Southeast Unversity
#### Project overview
Kubernetes's own scheduling algorithm is a scoring algorithm, its main purpose is load balancing, and the server cluster provider wants to improve resource utilization.
#### Main responsibilities
1, Read the source code of Kubernetes-scheduler, understand the Kubernetesscheduler extension interface, and extension. 2, A heuristic algorithm based on vector distance is proposed to improve resource utilization. 3, The overall arrangement of the algorithm to achieve the task.

### **4、Distributed IP address anti-collision assignment algorithm optimization**
Huawei Technologies Co., Ltd. and Southeast University
#### Project Overview
Base stations in a certain area can assign IP addresses to users, but when the user moves to another base station, the new base station must reallocate non-repeating new IP addresses, otherwise address collisions will occur. This project tries to reduce the overall collision probability.
#### Main responsibilities
1, The problem is abstracted as a mathematical model, and the method of using vertex coloring is proposed. 2, A distributed vertex coloring algorithm is proposed to reduce the number of base station communication and the number of color blocks required for coloring. 3, The establishment of a new IP data structure so that the probability of conflict is 0 when the base station is not busy, and the probability of conflict when the base station is busy will reduce. 4, Python to run simulation experiments, and in the final submission of C language code for use.
